# 《禁区：生命线》简介：
这是一个经营公司的小游戏，你可以在七天时间内招聘并培养员工，员工会通过异想体来获得报酬，快来把你的公司建设得越来越好吧！
这个项目完全由C++编写，目前推出1.0试测版本，这也是唯一的版本。

This project is written entirely in C++ and is currently available in beta 1.0, which is also the only version.
